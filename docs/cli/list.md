# Listing images

Much like the `docker images` command, the `policy` CLI allows you list local images:

```bash
policy list
```

## Example

```bash
$ policy list

|        REPOSITORY        |  TAG  | SIZE  |  CREATED AT  |
|--------------------------|-------|-------|--------------|
| omrigazitt1/peoplefinder | 1.0.0 | 555 B | 1 minute ago |
```
